Валидатор данных – библиотека, с помощью которой можно проверять корректность любых данных. 
Позволяет проверять:
  строки 
    не нулевое значение
    на минимальную длинну
    вхождение подстрок
  числа
    не нулевое значение
    на положительность
    вхождение в диапазон
  мапы
    не нулевое значение
    на размер
    данные внутри
